Autonomous Mobile Robot Simulation

I built a simulated navigation stack for an autonomous mobile robot using ROS 2 and Gazebo, focusing on reliable localization and obstacle-aware routing. The project paired behavior trees with the Nav2 stack and RViz visualizations so that the robot could plan, monitor, and update its course through a warehouse-style map.

Key contributions included tuning sensor models for lidar and depth cameras, configuring SLAM and AMCL pipelines for robust pose estimation, and wiring velocity commands into a differential-drive controller. I also automated map generation and waypoint missions with launch files, letting the robot trace repeatable inspection routes while reacting to new obstacles.

Navigation stack configuration

Configured Nav2 planners and controllers, balancing smooth trajectories with fast replanning. Added behavior-tree nodes for goal retries and collision recovery so the robot could escape tight aisles without human input.

Simulation and validation

Modeled lidar, IMU, and depth sensors to mirror real-world noise, then validated localization with loop-closure paths and deliberate obstacle insertions. Collected bag files to profile latency, CPU load, and controller stability for future hardware deployment.
